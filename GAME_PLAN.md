# Roguelite Prototype

We are building a small 3D roguelite prototype in Godot 4.

Goal:
A playable low-poly / PS1-style 3D roguelite vertical slice.

Milestone 1:
- Third-person player movement
- Camera follow
- Simple test room
- One enemy
- Basic attack
- Health/death/restart loop

Rules:
- Use GDScript.
- Keep code beginner-readable.
- Build one small feature at a time.
- After each feature, explain how to test it in Godot.
- Do not add complex systems unless asked.
